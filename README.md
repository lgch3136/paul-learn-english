# 译林版五年级下册英语同步陪练 MVP

## 项目目标

面向小学生开发英语辅导工具，重点解决：
- 单词记不住
- 语法记不住老错

**教材版本**：译林版小学英语（三年级起点）五年级下册，新课标版 / 2024 修订版 / 2026 春使用版

## MVP 范围

### 研发验证版：先做 1 个 Unit
- 1 个单元的单词、短语、核心句、语法点、文化点
- 单词练习：听音认词、看中文选词、拼写填空、句中应用
- 语法练习：选择、填空、改错、造句
- 错因记录：拼写错、词义混、句型错、语法错、粗心
- 复习计划：明天自动复习今天错得多的内容
- 家长日报：告诉家长今天错在哪里、明天怎么陪

## 核心功能

### 1. 单词记忆（5 个能力维度）
- 听得出：听音选图、听音选词
- 认得出：英文选中文、图片选英文
- 说得出：跟读、录音比对
- 拼得出：缺字母填空、完整拼写
- 用得出：放进句子、看图说句子

### 2. 语法错因诊断
错因标签体系：
- 主语判断错：I / you / he / she / they 分不清
- be 动词错：am / is / are 乱用
- 动词形式错：动词原形、三单、现在分词混淆
- 疑问词错：what / where / when / why / how 混淆
- because 用法错：why 问句不会用 because 回答
- 介词错：in / on / at / under / near / beside
- 时间表达错：月份、日期、序数词
- 句序错：中文顺序直译成英文
- 拼写错：单词漏字母、多字母、顺序错
- 读题错：题目要求没看清

### 3. 每日 10 分钟任务
- 2 分钟：复习昨天错词
- 3 分钟：学今天新词
- 3 分钟：做语法小题
- 2 分钟：跟读/造句输出

### 4. 复习算法
- 答对 2 次：隔 3 天再出现
- 答错 1 次：明天再出现
- 连续错 2 次：当天换形式再练
- 连续错 3 次：进入家长日报重点提醒

## 开发计划

### 第 1 周：教材核对 + 数据模板
- [ ] 拿到最新版五下教材
- [ ] 确认目录、Unit 标题、板块名称
- [ ] 完成 1 个 Unit 的知识拆解
- [ ] 建立单词表、语法表、题目表

### 第 2 周：学生端原型
- [ ] 完成今日任务
- [ ] 完成单词闯关
- [ ] 完成语法小练
- [ ] 记录答题结果

### 第 3 周：错因诊断 + 复习计划
- [ ] 每道错题能打错因标签
- [ ] 系统能生成明日复习
- [ ] 孩子能看到"哪里错了"
- [ ] 家长能看到日报

### 第 4 周：找孩子试用
- [ ] 找 5-10 个五年级孩子试用 5 天
- [ ] 记录每天完成率、错题减少率、家长反馈

## 关键指标

| 指标 | 目标 |
|------|------|
| 每日完成率 | 70% 以上 |
| 单次练习时长 | 8-12 分钟 |
| 重复错题下降 | 3 天内下降 |
| 家长看日报比例 | 60% 以上 |
| 孩子是否愿意继续 | 必须重点观察 |

## 重要提醒

⚠️ **教材版本核实**
网上很多"译林版五年级下册目录"仍然是旧版结构（如 Unit 1 Cinderella、Unit 8 Birthdays），但新版教材已经调整。
必须以最新版实体教材、官方电子教材或学校实际使用教材为准。
