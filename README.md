# 英语小达人 - 译林版小学英语同步陪练

## 项目简介

面向小学生的英语学习互动工具，通过游戏化方式提升单词记忆和语法掌握。
支持译林版小学英语三至六年级全部内容（含上下册），覆盖 400+ 单词、60+ 语法点。

## 核心功能

### 🎮 9 种单词练习模式

| 模式 | 图标 | 难度 | 说明 |
|------|------|------|------|
| 经典模式 | 📖 | 简单 | 看单词，选中文意思 |
| 听力挑战 | 🎧 | 简单 | 听发音，选出正确的单词 |
| 错题复习 | 🔄 | 简单 | 专门练习之前做错的题 |
| 闪电速度 | ⚡ | 中等 | 限时 6 秒，快速答题 |
| 单词钓鱼 | 🎣 | 中等 | 点击水中游动的正确鱼 |
| 字母拼拼乐 | 🔤 | 中等 | 把打乱的字母拼成单词 |
| 闯关挑战 | 🏆 | 困难 | 连续答对 8 题通关 |
| 拼写达人 | ✍️ | 困难 | 看中文意思拼出英文单词 |
| 单词弹射 | 🏹 | 困难 | 高端金属弩瞄准射击，直线弹道+墙面反弹，3D发光底座 |

### 📝 语法练习
- 正确/错误例句对比选择
- 详细语法要点解释
- 常见错误标签提示
- 按年级/单元筛选

### 🏆 成就系统
- **200+ 成就**，覆盖词汇积累、连击、答题总数、每日坚持等 13 个类别
- 等级系统：从 Lv.1 学徒到 Lv.20 传奇大师
- 积分奖励与称号收集
- 成就弹窗队列，避免信息爆炸

### 📊 学习统计
- 每日签到打卡
- 连续学习天数追踪
- 答题正确率/连击数/用时统计
- 家长查看学习报告

### 🎯 智能学习
- 基于答题表现的智能出题（question-scheduler）
- 错题自动归集复习
- 间隔重复算法
- 答题历史性能追踪

## 技术栈

- **框架**: Next.js 14 (App Router) + React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **动画**: CSS 动画 + Canvas 2D (弹射/钓鱼模式)
- **存储**: localStorage（无后端数据库）
- **语音**: Web Speech API（听力模式）

## 设计原则

- **趣味优先**: 每种模式都有独特的互动方式，避免单调重复
- **难度梯度**: 简单→中等→困难，循序渐进
- **即时反馈**: 音效、震动、动画、连击特效
- **家长友好**: 学习报告、签到追踪、成就激励
- **移动优先**: 响应式设计，触摸操作优化
