## 📱 如何在其他设备上访问

### 方法一：手机/平板访问

1. **确保设备连接到同一 WiFi**
   - 手机/平板和电脑必须在同一个局域网

2. **打开浏览器**
   - Safari（iPhone/iPad）
   - Chrome（Android）

3. **输入地址**
   ```
   http://192.168.2.24:3000
   ```

4. **开始使用！**
   - 可以添加到主屏幕方便下次访问

### 方法二：其他电脑访问

1. **确保连接同一网络**

2. **打开浏览器**

3. **输入地址**
   ```
   http://192.168.2.24:3000
   ```

## 📲 添加到手机主屏幕（推荐）

### iPhone/iPad
1. 用 Safari 打开 `http://192.168.2.24:3000`
2. 点击底部"分享"按钮 ⬆️
3. 选择"添加到主屏幕"

2. 点击右上角菜单 ⋮

4. 输入名称
5. 点击"添加"

## 🔧 常见问题

### Q1: 无法访问？
**检查：**
- ✅ 设备是否连接同一 WiFi
- ✅ 电脑防火墙是否允许（macOS 通常自动允许）
- ✅ 服务器是否正在运行

**解决：**
```bash
# 检查服务器是否运行
curl http://192.168.2.24:3000

# 如果无法访问，重启服务器
cd /Users/liugancheng/Documents/workbuddy/projects/paul-learn-english
npm run dev:lan
```

### Q2: 访问速度慢？
**原因：**
- 局域网设备过多
- WiFi 信号弱

**解决：**
- 靠近路由器
- 减少同时访问的设备

### Q3: 手机无法访问？
**检查：**
- 是否使用正确的地址：`http://192.168.2.24:3000`
- 注意是 `http://` 不是 `https://`
- 确保在同一 WiFi 网络
