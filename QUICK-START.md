## 🐛 常见问题

### Q: 没有声音？
A: 
1. 确保电脑/手机音量已打开
2. 点击页面任意位置激活音频
3. 检查浏览器是否允许音频播放

### Q: 动画卡顿？
A:
1. 使用 Chrome 90+ 或 Firefox 88+
2. 关闭其他占用资源的标签页
3. 检查电脑性能

### Q: 数据没有加载？
A:
1. 检查 `data/` 目录下的 JSON 文件是否存在
2. 检查 JSON 文件格式是否正确
3. 查看浏览器控制台是否有错误

### Q: 页面空白？
A:
1. 检查终端是否有错误信息
2. 确保 `npm install` 成功完成
3. 确保端口 3000 没有被占用
