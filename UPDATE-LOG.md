## 🚀 立即体验

### 启动项目
```bash
npm run dev
```

### 访问页面
- 今日练习：http://localhost:3000/lesson
- 单词闯关：http://localhost:3000/practice/vocabulary

### 体验流程
1. **今日练习** - 点击"一键开始练习"
2. **开启音乐** - 点击"开启背景音乐"按钮
3. **选择模式** - 选择"闪电速度"或"闯关挑战"
4. **获得成就** - 连续答对，解锁成就
5. **提升等级** - 积累积分，提升等级
